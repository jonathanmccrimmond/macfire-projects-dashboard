How leads are scored

Stage 1 — Regex

  • Checks for: new-build, change-of-use, restaurant, hotel, pub, gym, cinema, warehouse
  • Score: 0 – 100
  • Runs on every application
  • Instant, deterministic results
£0 cost

Stage 2 — AI (optional)

  • Runs only for borderline cases (score 20–45)
  • Model: configurable via AI_MODEL env var
  • Adds reasoning + confidence score
  • Swappable — any API-compatible model
~pennies/week
Model is swappable via env var — can use free alternatives without touching code
